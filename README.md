# SAR to Optical Image Translation

This repository contains code and pre-trained models for SAR to Optical image translation using deep learning.  
The project includes an encoder-decoder architecture with trained weights for inference.

---

## 📂 Project Structure

* app.py     [ Streamlit app for UI]
* decoder.py [ Decoder model definition]
* encoder.py [ Encoder model definition]
* inference.py [ Run inference: python inference.py]
* model.py [ Contains both Encoder and Decoder classes]
* decoder.pth [ Pre-trained decoder weights]
* encoder.pth [ Pre-trained encoder weights]
* model_1.pth [ Combined pre-trained model]


decoder.pth, encoder.pth and model_1.pth are big files and training files, So not uploaded here.
